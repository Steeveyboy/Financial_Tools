# S&P 500 Analysis

Jupyter notebooks for analyzing S&P 500 company fundamentals, including data collection from Yahoo Finance and visualization of key financial metrics.

## Notebooks

| Notebook | Description |
|----------|-------------|
| `S&P500_Stats.ipynb` | Statistical analysis and visualization of S&P 500 fundamentals |
| `tradingview_scraper.ipynb` | Scraping stock data from TradingView |

## Tech Stack

- **Data:** Pandas, yfinance
- **Visualization:** Matplotlib, Seaborn
