# Description Parsing

Jupyter notebooks for parsing and searching stock descriptions from NYSE-listed companies using Yahoo Finance data.

## Notebooks

| Notebook | Description |
|----------|-------------|
| `description_parsing.ipynb` | Parses company descriptions from financial data sources |
| `nyse_description_search.ipynb` | Searches and filters NYSE stock descriptions |

## Tech Stack

- **Data:** Pandas, yfinance
